
# BlissApplication Test Frontend




## Functionalities

- List of questions with pagination.
- Question details.
- Question quiz.
- Share the question link.


## Running locally

Clone the project

```bash
  git clone https://github.com/MalvesGO/bliss-app.git
```

Enter the project directory

```bash
  cd project folder
```

Install the dependencies

```bash
  npm install
```

Start the project

```bash
  npm run dev
```
